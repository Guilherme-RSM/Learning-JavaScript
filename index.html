Aprendendo JavaScript

Introdução ao JavaScript

A linguagem de programação JavaScript foi desenvolvida por Brendan Eich em 1995;
e a princípio era utilizado apenas como um recurso que deixava as páginas web mais interativas e era uma tecnologia Client-Side, Portanto rodava apenas do lado do cliente.

Inicialmente a missão de interpletar o JavaScript era apenas do navegador, até que o Engenheiro de Software, Ryan Dahl tivesse a brilhante ideia de pegar o interpletador V8, e trazelo para fora do ambiente do navegador, dando início então ao Node JS.
